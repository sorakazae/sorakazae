## 🚀 프로젝트 (プロジェクト)

### 📂 Project 1 - [MyGG]  
🖥 [Mygg GitHub Repository](https://github.com/SCIT46) | 🌎 [배포 링크・デプロイリンク（韓国語）](https://mygg.lol)  

**설명（説明）**  
이 프로젝트는 게임 "리그 오브 레전드"의 전적과 통계를 제공하는 것에 초점을 맞춘 웹 서비스입니다.  
このプロジェクトは、ゲーム「League of Legends」の戦績と統計を提供するWebサービスです。  

사용자가 게임 플레이 전적 조회를 쉽게 이용할 수 있도록 설계되었습니다.  
ユーザーがプレイ履歴を簡単に検索できるように設計されています。

**기술 스택（技術スタック）**  
- Backend: Java, Spring Boot, JPA, MySQL  
- Frontend: Thymeleaf, jQuery  
- DevOps: GitHub Actions, Docker  

**역할（役割）**  
- BackEnd 개발, DevOps 및 인프라 개발  
　→ バックエンド開発、DevOpsおよびインフラ構築  
- 아이템, 챔피언, 전적정보 조회 DB/API 설계 및 개발, 서버 및 CI-CD 파이프라인 구축  
　→ アイテム、チャンピオン、戦績情報のDB・API設計と開発、サーバーとCI/CDパイプライン構築

**성과（成果）**  
- 사용자가 조회하고자 하는 플레이어에 대한 게임 전적 검색 결과 제공  
　→ ユーザーが検索したいプレイヤーの戦績を表示  
- 아이템, 챔피언에 대한 정보 제공  
　→ アイテムやチャンピオンに関する情報の提供

### 📂 Project 2 - [UPDA]  
🖥 [UPDA GitHub Repository](https://github.com/SCIT46A) | 🌎 [배포 링크 (공사중)・デプロイリンク（現在は利用不可）](https://upda.store)  

**설명（説明）**  
UPDA는 환경 보호에 관심이 있는 창작자와 후원자를 연결하는 플랫폼입니다.  
UPDAは環境保護に関心のあるクリエイターと支援者をつなぐプラットフォームです。  

재활용 가능한 재료를 기부하고, 이를 활용한 제품 제작을 후원하며, 완성된 제품을 리워드로 받을 수 있습니다.  
リサイクル可能な材料を寄付し、それを使った製品作りを支援し、完成した製品をリワードとして受け取ることができます。

**기술 스택（技術スタック）**  
- Backend: Java, Spring Boot, JPA, MySQL  
- Frontend: Thymeleaf, jQuery  
- DevOps: GitHub Actions, Docker, Cloudflare  

**역할（役割）**  
- BackEnd 개발, DevOps 및 인프라 개발  
　→ バックエンド開発、DevOpsおよびインフラ構築  
- 캠페인 생성, 수정, 상품 생성관련 기능 및 DB 설계 및 개발  
　→ キャンペーンの作成・編集、商品管理機能、DB設計と開発  
- cloudflare images를 이용한 이미지 업로드 및 접근  
　→ Cloudflare Imagesを利用した画像のアップロードとアクセス制御  
- 서버 및 CI-CD 파이프라인 구축  
　→ サーバーとCI/CDパイプラインの構築
